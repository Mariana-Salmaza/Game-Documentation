Sobre o jogo



  • O jogo Enemy War é um jogo de Guerra que consiste em uma única fase.

  • O jogo é sigleplayer.

  • O jogo consiste em 1 jogador, 2 inimigos e um chefe.

  • No jogo haverá 1 perosnagem e 3 inimigos, um deles será o chefe, o mais forte do jogo.

  • O jogador terá que passar pelo os inimigos que haverá durante o jogo, matando-os, para que não seja derrotado.

  • No jogo haverá uma chave escondida para abrir um baú para pegar uma bomba.

  • No baú consiste uma bomba forte para você destruir o chefe, e assim terá a vitória do jogo.

  • os inimigos tem a função de tirar a vida do jogador conforme você relar neles, fazendo com que o jogo começe do ínicio.

  • O jogo consiste na sua atenção, para que o personagem principal não zera as suas 3 vidas ao deixar os inimigos relarem no seu jogador.



Ficha Técnica



  • Engine: Construct 3

  • Plataforma: PC

  • Mecânica: Plataforma

  • Dispositivos de entrada: Teclado



Elementos



  • Movimentação

  • Ataque corpo a corpo com espada

  • Quantidade limitada de vidas



Regras do jogo



  • Utilize as setas direita, esquerda, cima e baixo para se locomover.

  • Aperte a barra de espaço para fazer um ataque de espada para se defender dos inimigos.

  • Não deixe os inimigos relar em você para não perder as 3 vidas.

  • Encontre uma chave que está escondida para abrir o baú para pegar uma bomba que consiste dentro dela.



Personagem e inimigos



  • Jogador: Você

  • Inimigos: 2 porcos e 1 chefe.
